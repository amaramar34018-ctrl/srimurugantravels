# Travels Booking Website

## 🌍 Project Description
This is a full-stack travel booking website developed using Flask, HTML, CSS, JavaScript, and Bootstrap.

## 🚀 Features
- Travel package listing
- Booking form
- Responsive design
- Backend using Flask
- Hosted on Render

## 🛠️ Tech Stack
- Python (Flask)
- HTML
- CSS
- JavaScript
- Bootstrap
